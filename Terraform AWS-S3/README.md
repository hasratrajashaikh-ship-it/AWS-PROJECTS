
# Terraform AWS S3 Bucket

## Overview

This project uses **Terraform** to create an **AWS S3 bucket** and upload a file to it automatically.

## Technologies

* Terraform
* AWS S3
* Infrastructure as Code (IaC)

## Files

* `main.tf` – Terraform configuration
* `myfile.txt` – File uploaded to S3

## Steps to Run

1. Initialize Terraform

```
terraform init
```

2. Preview infrastructure

```
terraform plan
```

3. Create resources

```
terraform apply
```

## Result

* An S3 bucket named **demo-bucket7777** is created.
* The file **mydata.txt** is uploaded to the bucket.
